Demurrage Calculator Help

This page can be used to pay for multiple containers at one time as well as to pay demurrage through any date, whether for the current date or for any future date. It can also be used to calculate how much the demurrage charges will be due on any future date even if no payment will be made online. To pay for multiple containers for the current date or any future date:

1. Select or enter desired date (mm/dd/yy).
2. Click on the "Recalculate" button. The amount due will now be for the date selected.
3. Click in the check box to the left of each container to be paid (or click in the top most check box to select all)
4. Click on "Authorize Payment" Then follow the steps on the subsequent screen to complete the payment authorization.
5 If you do not wish to authorize a payment, please click Close to exit this page.

Please note: Each time the window repaints based on the user clicking on Recalculate, all checkboxes will return to an unpopulated state. If checkboxes are disabled, either there is no demurrage due for the date selected or that container's shipping line does not participate in the online payment process. In addition, if you have questions regarding how to use this Demurrage Calculator page, you can always use the "Contact Us" feature to send us a message that includes your question so we can further assist you.
